# JavaScript "Hello World" Program

This repository contains simple "Hello World" programs written in JavaScript.

## Files

- `hello-world.js` - Basic JavaScript file that can be run with Node.js
- `hello-world.html` - HTML file with embedded JavaScript for browser execution

## How to Run

### Option 1: Using Node.js (Command Line)
```bash
node hello-world.js
```

### Option 2: Using a Web Browser
1. Open `hello-world.html` in any web browser
2. Open the browser's developer console (F12) to see console output
3. Click the button to see an alert and page output

## Expected Output

When you run the JavaScript program, you should see:
```
Hello, World!
Hello, World!
```

In the browser version, you'll also see:
- Console messages in the developer tools
- Interactive button that shows an alert
- Dynamic content on the webpage

## About JavaScript

JavaScript is a versatile programming language that can run:
- In web browsers (client-side)
- On servers using Node.js (server-side)
- In mobile apps, desktop applications, and more

This simple "Hello World" example demonstrates the basic syntax and output capabilities of JavaScript.
