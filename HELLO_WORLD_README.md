# JavaScript "Hello World" Programs

This repository contains two different implementations of the classic "Hello World" program in JavaScript.

## Files

### 1. `hello-world.js` - Console Version
A simple Node.js console application that prints "Hello, World!" to the terminal.

**How to run:**
```bash
node hello-world.js
```

**Requirements:**
- Node.js installed on your system

### 2. `hello-world.html` - Web Browser Version
An interactive HTML page with embedded JavaScript that displays "Hello, World!" in a web browser.

**How to run:**
1. Open `hello-world.html` in any modern web browser
2. Click the "Click to Say Hello!" button to display the message
3. Check the browser's developer console to see console output

**Features:**
- Interactive button to trigger the message
- Styled with CSS for a better visual experience
- Automatic console logging when the page loads
- Responsive design

## About JavaScript

JavaScript is a versatile programming language that can run in web browsers and on servers (with Node.js). These examples demonstrate both environments:

- **Client-side**: The HTML file runs JavaScript in the browser
- **Server-side**: The .js file runs with Node.js on the command line

Both programs accomplish the same goal: displaying the traditional "Hello, World!" message that's often the first program written when learning a new programming language.
