## Simple Java Program

This repository now includes a simple Java program that demonstrates basic Java concepts.

### HelloWorld.java

```java
public class HelloWorld {
    public static void main(String[] args) {
        // Print a welcome message
        System.out.println("Hello, World!");
        System.out.println("Welcome to Java programming!");
        
        // Demonstrate variables
        String name = "Java Developer";
        int year = 2024;
        
        System.out.println("Hello, " + name + "!");
        System.out.println("Current year: " + year);
        
        // Demonstrate a simple calculation
        int a = 10;
        int b = 20;
        int sum = a + b;
        
        System.out.println("Sum of " + a + " and " + b + " is: " + sum);
        
        // Demonstrate a simple loop
        System.out.println("\nCounting from 1 to 5:");
        for (int i = 1; i <= 5; i++) {
            System.out.println("Count: " + i);
        }
    }
}
```

### How to Run

1. Save the code above in a file named `HelloWorld.java`
2. Compile the program:
   ```bash
   javac HelloWorld.java
   ```
3. Run the program:
   ```bash
   java HelloWorld
   ```
